# Depression vs CFS Classification

## Overview

This project presents a web-based application designed to classify whether an individual is more likely experiencing **Depression** or **Chronic Fatigue Syndrome (CFS)**. The system leverages a machine learning model integrated into an interactive interface to deliver real-time predictions.

---

## Objective

- Develop a classification model to distinguish between Depression and CFS  
- Deploy the model through an accessible web interface  
- Enable real-time user input and prediction  

---

## Application

The application allows users to input relevant data and obtain instant classification results based on the trained model.

🔗 **Live Web App:**  
https://depression-vs-cfs-classification-ganwzsrar3dwdcpe8j4ylt.streamlit.app/
